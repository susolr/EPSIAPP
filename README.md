# Práctica 0 y 1: Introducción a EPSIAPP

## ¿Qué es Epsiapp?

EPSIAPP es un sistema de alertas totalmente configurable destinado a personas que sufren Epilepsia.

## Mapa mental

<img src="https://github.com/susolr/EPSIAPP/blob/main/img/mapamental.png"/>

## Sistema a desarrollar

Son muchas las personas que sufren problemas cardíacos o determinadas enfermedades que condicionan el día a día de las mismas. En este caso queremos centrarnos en aquellas personas que padecen epilepsia en todos sus niveles, de forma que podamos facilitar su vida.

Para alcanzar este objetivo se llevará a cabo el desarrollo de un sistema software que permita al usuario que padece de epilepsia disponer de un sistema de alertas completamente configurable. Para ello, el usuario hará uso de relojes inteligentes o pulseras, de manera que sirva a la aplicación para monitorizar constantemente sus signos vitales y avisar en el caso en el que se detecte un ataque epiléptico. 

La aplicación permitirá también asignar tutores a determinadas personas, con el objetivo de acceder a los datos comentados anteriormente. Para ayudar al usuario, se permitirá almacenar un historial de ataques que ha padecido, así como recordatorios relativos a la medicación que deban tomar.
